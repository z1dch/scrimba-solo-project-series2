# 🚀 Scrimba Solo Project Series 2: Basketball Scoreboard

For the second project in the Scrimba solo series, I built a simple, interactive basketball scoreboard. This project marks my transition from static layouts into making web pages actually *do* something using **JavaScript**.

## What I Learned & Practiced

The main focus of this project was moving past HTML/CSS and dipping my toes into basic DOM manipulation. 

### Key Concepts Covered:
* **JavaScript Functions:** Learning how to write reusable blocks of code to handle repetitive tasks (like adding 1, 2, or 3 points to a score).
* **Working with `textContent`:** Understanding how to target an HTML element in JavaScript and update the text on the screen dynamically.
* **Event Handling:** Connecting HTML buttons to JavaScript functions so the score updates the exact moment a user clicks.

## How It Works

Instead of writing separate, messy code for every single button, I focused on using **functions** to keep things clean. When a user clicks "+1", "+2", or "+3" under the Home or Guest team, JavaScript catches the click, runs the math, and immediately updates the score display using `textContent`.

> 💡 **Side Note:** The design concept for the scoreboard layout was inspired by AI-generated UI ideas, which let me focus 100% on cracking the JavaScript logic rather than getting stuck on the visual design phase.
